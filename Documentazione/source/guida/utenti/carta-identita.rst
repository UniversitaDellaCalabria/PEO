Modifica documento di identità
==============================

.. include:: browser.rst

L'upload di un **documento di identità valido (fronte e retro)** è un requisito indispensabile per l'utilizzo degli strumenti messi a disposizione dalla piattaforma *peo.unical.it*.

In qualsiasi momento l'utente può aggiornare i file caricati, in autonomia, tramite la sezione dedicata (vedi menu laterale).

Il caricamento è limitato ai soli file con estensione *jpeg, png, gif, x-ms-bmp* e dimensione massima pari a 10 MB.


.. thumbnail:: images/documento_identita.png


**Come scansionare il documento con le "Kyocera Task Alfa"?**

In modo predefinito, il file viene salvato in formato pdf. 
Per evitare di incorrere nell'errore in fase di caricamento del documento, bisogna scegliere di salvare il file in formato JPEG, come da immagine allegata.
